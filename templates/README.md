# 🌍 [Flow] 
### المساعد التعليمي الذكي للاستدامة والتوعية البيئية 🚀

تطبيق ويب تفاعلي متكامل تم تطويره خلال الهاكاثون، يهدف إلى تعزيز الوعي بالاستدامة في قطاع التعليم باستخدام تقنيات الذكاء الاصطناعي (AI Agent) والمدخلات الصوتية، لتسهيل وصول الطلاب والمعلمين للمعلومات البيئية وتطبيقها بأسلوب ممتع وشامل.

🔗 **رابط التجربة الحية للتطبيق (Live Demo):** [اضغط هنا لفتح التطبيق وتجربته](ضعي_رابط_موقع_Render_الذي_ينتهي_بـ_onrender.com_هنا)

---

## ✨ مميزات التطبيق (Key Features)

- **شاشة بدء ذكية (Splash Screen):** واجهة ترحيبية سينمائية تستعرض هوية التطبيق وتنتقل تلقائياً لتهيئة المستخدم.
- **توجيه مبسط للمستخدم (Onboarding):** شاشة ترحيبية تفاعلية بزر "ابدأ الآن" تقود لنظام إنشاء حساب سلس ومقفل الثغرات.
- **شاشة الهوم والأيجنت الذكي (AI Agent Hub):** الواجهة الرئيسية للتطبيق والتي تحتوي على المساعد الذكي التفاعلي لتقديم حلول وأفكار تعليمية ومستدامة.
- **التسجيل والمدخلات الصوتية (Inclusive Voice Input):** واجهة مخصصة للتسجيل الصوتي المباشر لضمان تصميم شامل (Inclusive Design) يسهل تفاعل جميع فئات الطلاب.
- **شريط التنقل السفلي الذكي (Bottom Navigation Bar):** نظام تنقل مرن ومترابط بالكامل يتيح للمستخدم الانتقال بين (الرئيسية، التسجيل، حسابي) من أي مكان داخل التطبيق دون أي تعليق.
- **الملف الشخصي (User Profile):** واجهة مخصصة لعرض بيانات الحساب الشخصي ومتابعة الإنجازات البيئية.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

- **Backend:** Python & Flask Framework
- **WSGI Server:** Gunicorn (للإطلاق السحابي الدائم)
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
- **Design & Icons:** SVG Vector Graphics & Responsive Layouts
- **Deployment:** GitHub & Render Cloud Platform
