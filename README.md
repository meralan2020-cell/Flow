# 🌍 Flow 
### المساعد التعليمي الذكي لدعم الطلاب ذوي التأتأة وتعزيز دمجهم التعليمي 🚀

تطبيق ويب تفاعلي متكامل تم تطويره خلال الهاكاثون، يهدف إلى تحقيق **الاستدامة في التعليم والدمج الشامل** من خلال تقديم مساعد ذكي (AI Agent) مدعوم بتقنيات تواصل متطورة ومدخلات صوتية مخصصة، مصممة خصيصاً لتمكين ومساعدة الطلاب الذين يعانون من التأتأة أو صعوبات النطق، لتسهيل تواصلهم ومشاركتهم التعليمية بدون عوائق.

🔗 **رابط التجربة الحية للتطبيق (Live Demo):** [اضغط هنا لفتح التطبيق وتجربته](https://flow-hr8a.onrender.com)

---

## ✨ مميزات التطبيق (Key Features)

- **شاشة بدء ذكية (Splash Screen):** واجهة ترحيبية سينمائية تستعرض هوية التطبيق وتنتقل تلقائياً لتهيئة المستخدم.
- **توجيه مبسط للمستخدم (Onboarding):** شاشة ترحيبية تفاعلية بزر "ابدأ الآن" تقود لنظام إنشاء حساب سلس ومقفل الثغرات.
- **شاشة الهوم والأيجنت الذكي (AI Agent Hub):** الواجهة الرئيسية للتطبيق والتي تحتوي على المساعد الذكي التفاعلي لتقديم حلول وأفكار تعليمية ومستدامة، ومصمم ليتفاعل بصبر وسلاسة مع قدرات المستخدم.
- **التسجيل والمدخلات الصوتية الشاملة (Inclusive Voice Input):** واجهة مخصصة للتسجيل الصوتي المباشر، تم تصميمها لضمان تصميم شامل (Inclusive Design) يراعي فترات التوقف والنطق للطلاب الذين يعانون من التأتأة وتسهيل تفاعلهم.
- **شريط التنقل السفلي الذكي (Bottom Navigation Bar):** نظام تنقل مرن ومترابط بالكامل يتيح للمستخدم الانتقال بين (الرئيسية، التسجيل، حسابي) من أي مكان داخل التطبيق دون أي تعليق.
- **الملف الشخصي (User Profile):** واجهة مخصصة لعرض بيانات الحساب الشخصي ومتابعة التقدم التعليمي والتواصلي.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

- **Backend:** Python & Flask Framework
- **WSGI Server:** Gunicorn (للإطلاق السحابي الدائم)
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
- **Design & Icons:** SVG Vector Graphics & Responsive Layouts
- **Deployment:** GitHub & Render Cloud Platform
